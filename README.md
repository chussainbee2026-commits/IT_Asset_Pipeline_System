
# IT Asset Monitoring & Data Pipeline System

## Features
- Asset Management
- Employee Allocation
- Activity Logging
- CSV Report Generation
- Dashboard

## Technologies Used
- Python
- Flask
- SQLite
- Pandas

## Installation

1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
